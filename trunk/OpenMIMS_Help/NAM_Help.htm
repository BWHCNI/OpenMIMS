NRIMS Analysis Module

 

Introduction

The NRIMS Analysis Module is an ImageJ plugin to open, process and analyze images captured with the National Resource for Imaging Mass Spectrometry’s (NRIMS) Multi-isotope Imaging Mass Spectrometer (MIMS) system (http://www.nrims.harvard.edu/).  Images or image stacks of up to 6 different isotopes can be opened, each mass as a separate stack and saved. All Meta data saved in the original file are preserved and saved.  Image ratios and Hue-Saturation-Intensity (HSI) maps of any combination of isotopes can be displayed and data from any number of Regions of Interest (ROIs) extracted and tabulated for single images or entire stacks.

 

Installation

After downloading the Open_MIMS.jar (java archive), copy to the ImageJ home plugins folder and restart ImageJ.  Note this plugin requires a 1.36 or later version of ImageJ with the JRE 5.0 bundle (http://rsb.info.nih.gov/ij/download.html and click on the link “available bundled with JRE 5.0”).

 

Opening MIMS images

After launching ImageJ, place the mouse over the Plugins menu option and select the “Open MIMS->Open MIMS image” option.  This action will open a file dialog window. Navigate to the folder where the MIMS image (file with a .im extension) you wish to open was copied, select the file and click the Open button.  After the MIMS image is opened, each mass will appear in a separate window as well as the NRIMS Analysis Module window. 

 

You can open a new MIMS image by selecting the File->Open option in the NRIMS Analysis Module window.  If you modify any of the mass images through any of ImageJ’s processing functions, the original data can be reloaded by selecting NRIMS Analysis Module’s menu option Edit->Restore. To view all Meta data saved with a MIMS image, select ImageJ’s menu option Image->Show Info.

 

The NRIMS Analysis Module includes 3 tabs:

·        MIMS Data

Displays a subset of the images Meta data including the file name, number and AMU of masses including in the file, number of images per mass, position, User name, dwell time and raster size. The Synchronize Stacks check box enables updating all masses simultaneously when a MIMS image with multiple samples per mass has been opened.  Adjusting the slider (slice selector) at the bottom of any mass automatically selects the same slice for all masses,

·        Process

Selects parameters for displaying ratio images and HSI maps,

·        Analysis

Draw Regions of Interest (ROIs) and tabulate extracted statistics from one or more mass images and ratios.

Ratio and HSI Calculations

 

To compute a display the ratio of a pair of masses, first select the Process tab and then the mass number for the Numerator and Denominator image pair, and press the Display Ratio button. The ratio image will appear in a separate window. Ratio images are computed and displayed with 32 bit floating point precision with the contrast automatically adjusted to the minimum and maximum ratios.  When moving the mouse pointer over the ratio image, the status box along the lower border of the NRIMS Analysis Module window displays the raw numerator and denominator counts as well as the ratio.

 

To display an HSI map, select the range of ratios or press the Auto button to automatically scale the HSI image to the range from the raw counts.  The Threshold option set the minimum number of counts in the numerator and denominator. Any pixels with fewer counts will be ignored and appear black.  Pressing the Display HSI button opens an HSI map for the corresponding Numerator: Denominator mass pair.  After opening the HSI map, the Contrast slider can be adjusted to increase or reduce the color saturation. The Transparency selects the method for computing the intensity component of the HSI image, and the Label option enables a color scale bar for the ratio or hue component.

 

If the MIMS image has multiple samples per mass, adjusting the slider at the bottom of any of the mass images automatically updates the ratio image or HSI map to the corresponding slice.

Selecting ROIs

Select the Analysis tab,

 

and enable the Synchronize ROIs check box.  Next select the type of ROI to draw by clicking one of the corresponding toolbar buttons in ImageJ’s main window (any of the seven buttons on the left). Place the mouse over the area and drag the mouse while holding the left mouse button down. When the left mouse button is released, the corresponding ROI will appear in all masses.  If the Add ROIs check box is enabled, ImageJ’s ROI manager will appear and the ROI name will be added to any previously drawn ROIs.  CAUTION, if the ROI Manager window is closed before saving a list of ROIs, the drawn regions will be discarded. Clicking on a previously drawn ROI in ROI Manager’s list will automatically redraw the selected ROI in all masses, ratio images and HSI maps.

 

 

Data Analysis

To tabulate the extracted data from a series of ROIs, go to the NRIMS Analysis Module’s Analysis tab and press  the Source images button to select the masses to be included in the tabulated results. All open ratio images will automatically be included. To exclude a ratio pair, simply close the corresponding ratio image.

 

Pressing the Data Options button displays the following dialog

 

 

Enter the name of the file to save the tabulated data in the Table Name field. By default, the file name of the source MIMS image is used with the extension .txt. When Measure All Rois is pressed, the tabulated data is extracted displayed in an ImageJ results window. To save the tabulated results to a file, select the results’ window File->SaveAs menu option. The data is saved column major with all regions for the first mass written followed by all regions for the second mass for all masses and finally for all ratios. With multiple slices per mass, enabling the Measure All Sections check box cycles through each image in a stack and tabulates the results with each slice appended on a new row in the output. Each column has a header indicating the data source.

 

Geometric measurements (independent of counts or ratio values) are tabulated once per row. All other measurements are tabulated for each selected mass and ratio image.
